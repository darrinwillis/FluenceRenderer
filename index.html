Fluence Renderer Project Report

Darrin Willis - dswillis

This project implemented the infrastructure of a 2D ray-tracer.

Initially, it had fairly lofty goals of reproducing (this project)[https://benedikt-bitterli.me/tantalum/], however that ended up indeed being too lofty.
